# web 爬虫示例
使用Java实现爬取网页功能，代码非常简单，只是引入了jsoup 依赖包。内部包含工作和学习中用到的爬虫代码。此项目仅为了减少重复的机械劳动，重复劳动通过爬虫帮我们完成。

## pom配置
~~~xml
<dependency>
  <groupId>org.jsoup</groupId>
  <artifactId>jsoup</artifactId>
  <version>1.12.1</version>
</dependency>
~~~
